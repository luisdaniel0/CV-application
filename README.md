# CV Application (React)

The application allows users to:

- Enter **general information** (name, email, phone number)
- Add **educational experience** (school name, title of study, date of study)
- Add **practical experience** (company name, position title, main responsibilities, work dates)

Each section includes **Edit** and **Submit** functionality:

- **Submit:** saves and displays user input as formatted text
- **Edit:** re-enables the input fields with previously entered values for updating
